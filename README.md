PIZZA PLACE SALES ANALYSIS
PROJECT OVERVIEW
This project analyzes one year of sales data for a fictitious pizza place. The goal is to extract insights into sales trends, customer behavior, and product performance to help improve business operations.

DATASET
The analysis uses four CSV files linked by relational keys:
orders.csv: Date and time of all orders.
order_details.csv: Specific pizzas and quantities for each order.
pizzas.csv: Size and price data for each specific pizza variation.
pizza_types.csv   : Names, categories, and ingredients.

 TOOLS USED
Python: Primary programming language.
Pandas: For data manipulation and aggregation.
Jupyter Notebook: For interactive analysis and reporting.
Matplotlib : For data visualisation
